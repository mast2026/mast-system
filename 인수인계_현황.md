## 4. 핵심 구조 메모
- 활동날씨 점수: `clamp(70 + 수동가감 + 출석항목 + 에타미참여, 0~100)`. 출석/지각/에타미참여는 DB 저장 없이 기록에서 실시간 계산. 지각 감점은 OT 모임만, 10분 이내 출석.
- 테이블: 로그인/팀매칭 = `team_matching_*`, 출석/홍보 = legacy `members`(id 체계 다름, 이름·학교·기수 매핑).
- PWA: 회원/관리자 매니페스트 분리(ManifestSwitcher, /admin 경로 전환).
