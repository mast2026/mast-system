  - iOS는 **홈 화면에 PWA 추가 + iOS 16.4+** 필요. 안드로이드/PC는 브라우저 알림 허용이면 됨.

## 4. 핵심 구조 메모 (참고)

- **점수(활동날씨):** `clamp(70 + 수동가감 + 출석항목 + 에타미참여, 0~100)`. 출석/지각/에타미참여는 **DB에 저장 안 하고 기록에서 실시간 계산**(`activityWeatherService.js`의 DERIVED_EVENT_TYPES 제외). 지각 감점은 **OT 모임만**, 10분 이내는 출석.
- **테이블:** 로그인/팀매칭 = `team_matching_*`, 출석/홍보 = legacy `members` 테이블(id 체계 다름, `matchLegacyMember`/이름·학교·기수로 매핑).
- **PWA:** 회원/관리자 매니페스트 분리(`ManifestSwitcher`가 /admin 경로로 전환).
- **검증 팁:** JSX는 sandbox에서 vite/esbuild 빌드 불가(네이티브 바이너리 OS 불일치). 브레이스 밸런스 + `@babel/core` parseSync로 문법 확인함. 실제 빌드는 로컬에서.
- **동명이인 로그인:** `박정현`(연세대/단국대), `이수민`(부산가톨릭대/이화여대) → 학교·기수 드롭다운으로 구분.
